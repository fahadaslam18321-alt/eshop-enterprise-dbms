# eShop Enterprise DBMS Dashboard

A full-stack database management interface built to execute, test, and monitor complex relational database operations in real-time.

## 🔗 Live Demo
Check out the live interactive dashboard running on Hugging Face Spaces: 
👉 [eShop Frontend Live Demo](https://huggingface.co/spaces/fahadaslam18321/eshop-frontend)

## 🛠️ Tech Stack & Features
- **Frontend/Backend:** Python (`app.py`), Streamlit
- **Database Architecture Components:** 
  - **6 Stored Procedures:** Efficient data filtering and operations (e.g., fetching user info, inserting new records).
  - **7 SQL Views:** Structured aggregation of business data.
  - **5 SQL Joins:** Advanced relational queries connecting multiple datasets.
  - **Transactions & Triggers:** Automated integrity logic for sales and orders.
- **Data Layer:** Managed using structured relational CSV tables (Users, Products, Orders, Payments, Discounts, Reviews, etc.).
